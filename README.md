# HR-Attrition-Excel-Dashboard

The HR Analytics Dashboard is an interactive Excel dashboard developed to analyze employee attrition, workforce demographics, education, job roles, and other HR metrics. The objective of this project is to help HR managers and business leaders identify the major reasons behind employee attrition and make data-driven decisions to improve employee retention.

The dashboard was built using Microsoft Excel, Pivot Tables, Pivot Charts, Slicers, and advanced Excel functions to convert raw HR data into meaningful business insights.
